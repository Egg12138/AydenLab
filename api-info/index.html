ⅰ作息和生活篇


\n/作息

\n

/生活噪声

\n
4.床具噪音，肖和黄的宿舍床因结构问题噪声较大，需限制上下床动作幅度：\n    不得全掌贴地跳跃下床\n    上床事人体重心尽量偏向过道\n**有必要对门进行维修，门的噪声过大**\n5.有宿舍成员睡眠时，某些行为需受限，包括但不限于：\n开柜幅度减小、开拉门速度减慢、卫生间闭门开水、底噪走路、湿拖鞋走路不用全掌直接着地的方式、不在阳台和卫生间之外的场所洗漱。    \n

/生活洗漱

\n

/水电物质

\n宿舍（含卫生间和阳台）保洁、宿舍垃圾倾倒、堆积杂物（快递盒等）定期清理、宿舍气味。

/垃圾卫生
