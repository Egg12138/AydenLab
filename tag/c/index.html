C标准const限定：

         关键字const对变量的类型加以限定，使它的值不能被改变。
          const对象一旦创建后其值就不能再改变，所以const对象必须初始化，如下：
  const int i = get_size();  //正确，在运行时初始化
  const int j = 42;  //正确，在编译时初始化
  const int k;  //不正确，k是一个未经初始化的常量
